¿Qué diferencias y semejanzas hay entre Java y JavaScript?

JavaScript se integra directamente en las páginas HTML y es interpretado (sin estar compilado) por el cliente (navegador).[1]
JavaScript está basado en objetos, mientras que Java es un lenguaje de programación orientado a objetos (OOP). JavaScript no utiliza clases, herencias o técnicas habituales en la OOP.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • getDate();
    • Regresa el día del mes (1-31)
  • getDay();
    • Regresa el día de la semana (0-6)
  • getFullYear();
    • Regresa el año
  • getHours();
    • Regresa la hora (0-23)
  • getMinutes();
    • Regresa los minutos (0-59)

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • concat();
    • Junta dos o más arreglos y devuelve una copia de los arreglos juntados
  • every();
    • Revisa que todos los elementos de un arreglo pasen alguna prueba
  • fill();
    • Llena todos los elementos de un arreglo con un vsalor estático
  • find();
    • Regresa el primer valor del arreglo que pase alguna prueba
  • indexOf();
    • Regresa el índice del valor buscado en un arreglo
