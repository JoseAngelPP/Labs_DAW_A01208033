Preguntas a responder

¿Cuáles son las diferencias entre los posibles valores de la propiedad position?

  • Static: Un elemento con este valor de posición está posicionado con el flujo normal de la página.
  • Relative: Esta posición, como su nombre lo dice, hace que el elemento se fije relativo a su posición normal.
  • Fixed: Con este valor el elemento se queda fija en un ligar sin importar que la página sea desplazada.
  • Absolute: Este valor de posición deja al elemento posicionado relativo a su elemento padre.
  • Sticky: Este valor sirve para hacer un elemento cambiar entre fixed y relative pues se queda fijo después de haber desplazado la página.

¿Cuáles son los valores estándar para la propiedad visibility?

  • Visible: El elemento está visible.
  • Hidden: El elemento está escondido.
  • Collapse: Sólo funciona con tablas, y quita una columna o fila pero no afecta el layout.
  • Initial: Pone el valor default.
  • Inherit: Hereda el valor de su elemento padre.

¿Qué es el zIndex y para qué sirve?

Esta propiedad te permite poner capas en tu html. Le da profundidad y te permite mover elementos atrás de otros o enfrente de otros.
