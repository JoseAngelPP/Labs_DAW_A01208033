  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • En primer lugar que JavaScript es un lenguaje interpretado y Java es compilado. Otra diferencia es que Java se depura en dos fase y Java Script en solo una. La mayor diferncia es que Java Script funciona para darle comportamiento a las paginas web, mientras que Java es un lenguaje orientado a objetos.

  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    • getDay (Método): Devuelve el valor de día de la semana usando la hora local.
    • getFullYear (Método): Devuelve el valor de año usando la hora local.
    • getHours (Método): Devuelve el valor de horas usando la hora local.
    • getDate (Método): Devuelve el valor de día del mes usando la hora local.
    • getMinutes (Método): Devuelve el valor de minutos usando la hora local
  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
    • Array.from(): Crea un nuevo Array de un objeto iterable o parecido a un array.
    • Array.prototype.pop(): Elimina el último elemento de una matriz unidimensional y retorna este elemento
    • Array.of(): Crea un nuevo Array con un numero de argumentos variable, sin importar numero o tipo de argumentos.
    • Array.isArray(): Devuelve true si una variable es un Array, de lo contrario devuelve false.
    • Array.observe(): Observa los cambios del Array asincrónicamente, similar a Object.observe() para los objetos. Provee un flujo de cambio en orden.

  • ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?

    • - match: Un método String que ejecuta una búsqueda por una coincidencia en una cadena. Devuelve un array de información o null si no existe coincidencia alguna.
    - search: Un método String que verifica una coincidencia en una cadena. Devuelve el índice de la coincidencia, o -1 si la búsqueda falla.
