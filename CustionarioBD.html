PREGUNTAS

  • ¿Qué es entonces una base de datos - DB?

    • Es cualquier conjunto de datos o información organizado para facilitar la búsqueda y recuperación rápida por un ordenador. Las bases de datos están planeadas con el fin de facilitar el almacenamiento, recuperación, modificación y eliminación de datos.

  • ¿En qué casos es conveniente usar bases de datos?

    • Cuando un sistema que almacena información requiera de las capacidades de referencia. Las bases de datos permiten a los usuarios buscar, reorganizar datos de manera rapida.

  • ¿Qué es un sistema de gestión de base de datos - DBMS?

    • Es un Sistema para la búsqueda rápida, ademásd de ayudar a la recuperación de información de una base de datos.
